About the Project

Communication when
networks fail

The NLR Disaster Communication System V1 is a prototype designed to validate offline multi-user communication in disaster scenarios using a single ESP32 microcontroller.

When cellular towers fail and internet connectivity is cut, this system creates its own local WiFi network — enabling rescue teams, volunteers, and survivors to exchange real-time messages through any standard smartphone browser.

Developed by NLR Group of Company, Version 1 focuses on proving the core concept: offline communication is achievable at minimal cost with commodity hardware.

Functional Features

What V1 delivers

A focused set of features that proves the offline communication concept works reliably and simply.

📡
WiFi Hotspot
ESP32 broadcasts SSID Disaster_Network with password protection. No router needed.
🌐
Web Server on Port 80
Hosts a lightweight HTML interface accessible at 192.168.4.1 from any connected device.
💬
Real-time Chat
Message input, send button, and auto-refreshing chat display. Messages update every 1 second across all connected users.
🚨
SOS Alert Highlighting
Messages containing SOS are automatically highlighted in red with priority styling for immediate visibility.
👥
Multi-user Support
Multiple users can connect simultaneously and exchange messages in real-time via shared RAM-based message store.
⚡
REST API Endpoints
Three clean HTTP endpoints: GET /, GET /send, GET /get for web and app integration.

// NLR DisasterComm V1 — Core Sketch
#include <WiFi.h>
#include <WebServer.h>

const char* ssid = "Disaster_Network";
const char* password = "12345678";

WebServer server(80);
String messages[50];
int msgCount = 0;

void setup() {
  WiFi.softAP(ssid, password);
  server.on("/", handleRoot);
  server.on("/send", handleSend);
  server.on("/get", handleGet);
  server.begin();
}

void loop() {
  server.handleClient();
}

void handleSend() {
  String msg = server.arg("msg");
  if (msgCount < 50)
    messages[msgCount++] = msg;
  server.send(200, "text/plain", "OK");
}

void handleGet() {
  String resp = "";
  for (int i=0; i<msgCount; i++)
    resp += messages[i] + "\n";
  server.send(200, "text/plain", resp);
}
